Publications and Projects

LiDAR Distillation: Bridging the Beam-Induced Domain Gap for 3D Object Detection
Yi Wei, Zibu Wei, Yongming Rao, Jiaxin Li, Jiwen Lu , Jie Zhou
European Conference on Computer Vision (ECCV), 2022
[arXiv] [Code]

We propose the LiDAR Distillation to bridge the domain gap induced by different LiDAR beams for 3D object detection.

Smart Explorer: Recognizing Objects in Dense Clutter via Interactive Exploration
Zhenyu Wu*, Ziwei Wang*, Zibu Wei , Yi Wei, Haibin Yan
IEEE/RSJ International Conference on Intelligent Robots and Systems (IROS), 2022
[arXiv] [Demo] [Code]

We propose an interactive exploration framework called Smart Explorer for recognizing all objects in dense clutters.

Versatile Lab-Cruising Robot
Zibu Wei, Zhuoshi Pan, Chentao Li, Dongyang He
Applied on the 5th floor in the Main Building of Tsinghua (Labs of Department of Automation)
Advised by Prof. Yanpin Ren

We design a versatile lab-cruising robot based on Robot Operating System (ROS), which has great functions, including speech interaction, photography and video recording, temperature measurement, SLAM and navigation, remote control, etc.

Facial Expression Recognition
Project of Foundation of Artificial Intelligence
Advised by Prof. Rui Jiang

The project is to build a deep network for facial expression recognition using pytorch. Moreover, by writing an interface for inputting images and video, we achieve the automatic and real-time face recognition and expression recognition.

LianLianKan Small Game
Project of Foundation of Artificial Intelligence
Advised by Prof. Rui Jiang

The project is to utilize searching algorithms to eliminate icon pairs with fewer turns. We use greedy algorithm and breadth-first-search algorithm in this project.
