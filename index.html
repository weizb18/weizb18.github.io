Object Retrieval and Shape Completion from Multi-view RGBD Images of Dense Clutter
Zibu Wei, Yi Wei, Ziwei Wang, Zhenyu Wu, Jiwen Lu , Jie Zhou
In progress. Plan to submit to CoRL2022
Coming soon.

We propose a pipeline to retrieve objects and complete shapes from multi-view RGBD images of dense clutter. We recognize every object in 2D and further generate objects' incomplete point clouds by matching the corresponding point clouds under each view. Then, we input objects' point clouds into an encoder and get the query features. By retrieving the query feature in template features database, we get the most similar template and complete the object shape.

LiDAR Distillation: Bridging the Beam-Induced Domain Gap for 3D Object Detection
Yi Wei, Zibu Wei, Yongming Rao, Jiaxin Li, Jiwen Lu , Jie Zhou
Submitted to ECCV2022

We propose a progressive framework to mitigate the domain gap induced by different LiDAR beams for 3D object detection. Low-beam pseudo LiDAR is generated by downsampling high-beam point clouds. Then the teacher-student framework is employed to distill rich information from high-beam data. Experiments on Waymo, nuScenes and KITTI datasets demonstrate the effectiveness of the method.

Smart Explorer: Recognizing Objects in Dense Clutter via Interactive Exploration
Zhenyu Wu*, Ziwei Wang*, Zibu Wei , Yi Wei, Haibin Yan
Submitted to IROS2022

We propose an interactive exploration framework for recognizing all objects in dense clutters based on multi-view RGB-D cameras. The robot pushes the clutter to reduce the prediction uncertainty with spatial relation constraint, which significantly enhances the recognition accuracy. Experiments indicate that our Smart Explorer outperforms random pushing by a large margin and sizably increases the recognition accuracy with only a few actions.

Facial Expression Recognition
The AI course project
Advised by Prof. Rui Jiang

The project is to build a deep network for facial expression recognition using pytorch. We train the ResNet on the expression dataset. With an interface for inputting images and video, the project automatically recognizes the faces and estimates the expressions.

LianLianKan Small Game
The AI course project
Advised by Prof. Rui Jiang

The project is to design the game using searching algorithm. We aim to eliminate econ pairs by fewer turning times. The project use greedy algorithm and breadth-first-search algorithm to solve the problem, and we analyze the potential methods of A* algorithm and backtracking algorithm.
