Lab-Cruising Robot
Zibu Wei, Zhuoshi Pan, Chentao Li, Dongyang He
Applied on the 5th floor in the Main Building of the Department of Automation
Advised by Prof. Yanpin Ren

We design a lab-cruising robot based on Qizhi ROS Platform. The robot has great functions(Speech Interaction, Photography and Video Recording, Temperature Measurement, SLAM and Navigation, Remote Control, etc.) and it has been applied to introduce the labs to visitors.

Facial Expression Recognition
The AI course project
Advised by Prof. Rui Jiang

The project is to build a deep network for facial expression recognition using pytorch. We train the ResNet on the expression dataset. With an interface for inputting images and video, the project automatically recognizes the faces and predicts the expressions.

LianLianKan Small Game
The AI course project
Advised by Prof. Rui Jiang

The project is to solve the game using searching algorithm. We aim to eliminate icon pairs with fewer turns. The project uses greedy algorithm and breadth-first-search algorithm to solve the problem, and we further analyze the potential solutions, such as A* algorithm and backtracking algorithm.
